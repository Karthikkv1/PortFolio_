Simple Java Chat Application using Socket Programming concept

I have developed a simple Java Chat Application to showcase Socket programming in Java. This application allows users to connect over a network and exchange messages in real-time. It features a basic client-server architecture where one user acts as the server and others as clients. The server handles incoming client connections and facilitates message broadcasting among connected clients. Users can send and receive messages seamlessly through a console-based interface. The application demonstrates fundamental concepts such as socket creation, binding, listening, accepting connections, and data communication using input/output streams. It serves as a practical example for understanding network communication in Java programming.
